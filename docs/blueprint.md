# **App Name**: WanderWise AI

## Core Features:

- AI Travel Planning: AI-powered trip planning that generates personalized itineraries based on user inputs.
- Quick Planner Form: A user-friendly form to input trip details, preferences, and constraints.
- Itinerary Preview: Show a sample day-by-day itinerary structure using an accordion.
- Travel Timeline: Display past and upcoming trips in a horizontally scrollable timeline.
- Firebase Authentication: Secure user authentication using Firebase Authentication.
- Side Assistant Widget: Integrate a circular, floating action button that serves as a chatbot entry point for user support and assistance; implement AI to respond to the user.

## Style Guidelines:

- Primary color: Muted green (#6A8D73) for primary buttons, active links, and important UI elements, evoking a sense of nature and tranquility.
- Background color: Beige (#F2EFE5) for page and card backgrounds, providing a soft and calming backdrop.
- Accent color: Gold (#C9A66B) for icons, borders, and subtle highlights, adding a touch of elegance and sophistication.
- Font pairing: 'Playfair' (serif) for headlines and small amounts of text to evoke elegance, with 'PT Sans' (sans-serif) for body to improve readability for larger text.
- Use clean, minimalist icons with the accent gold color to represent various travel-related actions and categories.
- Implement a clean, minimal, and mobile-first responsive design using Flexbox and Grid, with rounded-2xl corners for all cards, forms, inputs, and buttons, and soft shadows (e.g., shadow-md) on elevated elements.
- Subtle animations for transitions and loading states to enhance the user experience and provide feedback.
